Technical Description

CSS techniques included Flexbox for responsive layout, CSS animations for the countdown timer, and media queries for mobile responsiveness.

Challenges and Lessons Learned

One of the main challenges was ensuring cross-browser compatibility. This was resolved by testing the pages on multiple browsers and using CSS prefixes where necessary. Another challenge was implementing responsive design, which was addressed using media queries and flexible layouts. Through this project, I learned the importance of planning and testing early in the development process.

Compliance

The pages comply with Janet regulations by ensuring all content is accessible, using secure protocols for external links, and providing clear navigation and user feedback.
