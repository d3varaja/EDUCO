Challenges and Lessons Learned


Responsive Design Implementation

One difficult challenge was creating a consistently responsive designs for the home,Table and content pages particularly the card sections. First, the cards would break layout on smaller screens, overlapping or displaying incorrectly.

Solution:

I implemented a combination of flexbox with the flex wrap property and media queries targeting specific breakpoints. This allowed cards to reflow naturally on smaller screens while maintaining their visual integrity. I learned the importance of testing across multiple device sizes throughout development..


CSS Animation Performance

When implementing hover effects for the mission and vision sections, I encountered performance issues where animations would stutter on some mobile devices.

Solution:

I optimized the animations by using CSS transform properties instead of animating dimensions directly, as transforms are more accelerated.In here I learned i have to be more mindful when designing interactive elements, especially for mobile devices with limited processing power.


Managing Content Load Time

Large images and content-heavy sections caused the page to load slowly, particularly for users with slower internet connections or mobile data.

Solution:

To optimize content load times, I compressed and resized images in the home page cards without compromising on quality, using tools like ImageOptim or TinyPNG. I learned here that the large images take more time to load the pages.
